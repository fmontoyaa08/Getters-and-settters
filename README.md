# Tarea: Encapsulamiento en Java (Getters y Setters)
**Estudiante:** Fátima Montoya
**Institución:** Universidad Mariano Gálvez de Guatemala

##  Descripción
Este proyecto contiene la resolución de 10 ejercicios prácticos enfocados en el concepto de **encapsulamiento** en Programación Orientada a Objetos. Se aplicaron modificadores de acceso `private`, validaciones en métodos `set` y lógica de negocio para gestionar datos de forma segura.

##  Estructura del Proyecto
El proyecto está organizado en el paquete `TareasEncapsulacion` y cuenta con las siguientes clases:

1. **Persona:** Manejo de datos básicos.
2. **Producto:** Gestión de stock y retiros.
3. **Cuenta:** Sistema de seguridad con PIN y saldo.
4. **Curso:** Validación de notas académicas.
5. **Estudiante:** Cálculo de promedios y estado de beca.
6. **Termómetro:** Conversión de Celsius a Fahrenheit.
7. **Reloj:** Formato de tiempo HH:MM:SS.
8. **Contacto:** Gestión de números telefónicos.
9. **Jugador:** Récords de puntaje máximo.
10. **Factura:** Cálculo de totales de venta.
